# County Line Crew App Plan

County Line Crew is a mobile-first scheduling and invoice app for mowing, cleanup, handyman, and property work.

## Goal

Build a simple app that lets a customer request work and lets the owner manage jobs, customers, payments, and invoices from a phone.

## First Version

The first version should stay simple and useful:

- Add customers
- Add jobs
- Track job date, location, service, notes, and price
- Mark jobs as scheduled, completed, paid, or unpaid
- View a basic invoice screen
- Store data locally in the browser at first

## Target Users

### Customers

Customers should be able to:

- Request mowing, cleanup, handyman, or property work
- Enter name, phone, email, and address
- Describe the job
- See confirmation after submitting

### Owner/Admin

The owner should be able to:

- View all jobs
- Add or edit customers
- Add or edit jobs
- Track paid and unpaid work
- Pull up invoice details from a phone

## App Style

The design should be:

- Phone-first
- Simple
- Fast
- Large buttons
- Easy to use outdoors
- Clean enough for customer-facing screens
- Practical enough for daily work

## Build Approach

Start with plain HTML, CSS, and JavaScript. Add local browser storage first. Add a backend later only after the basic flow works.
